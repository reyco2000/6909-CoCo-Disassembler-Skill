Look up quick reference information for CoCo 6809 development. Read and use the files in the `reference/` directory to answer questions about:

- **Memory map**: `reference/coco_memory_map.md` — addresses, PIA, SAM, VDG, FDC, vectors
- **ROM calls**: `reference/rom_entry_points.md` — Color BASIC, Extended BASIC, Disk BASIC entry points
- **Opcodes**: The opcode tables in `tools/dasm6809.py` (OPCODES_PAGE1, PAGE2, PAGE3)

When answering, always provide:
1. The hex address
2. The symbolic name
3. A brief description of what it does
4. Example usage in 6809 assembly if relevant

$arguments: topic — what to look up (e.g. "POLCAT", "PIA1", "joystick", "interrupt vectors")
